# Weather App 

Weather forecast demo app.

## Features

- see forecasts for each locations
- save forecasts results
- see forecasts on a location for the rest of the week
- dynamically search or create locations

## Requirements

- Rails 7.1.3.2 
- Ruby 3.1.0p0
- sqlite3

## Installation

1. Clone This repo.
2. Install the gems:

   ```sh
   bundle install
   ```
3. migrate db:
   ```sh
   rails db:migrate
   ```

   
## Run Test
- run `rails test`
